# 大作业报告提纲

## 1. 研究背景

介绍强化学习与多智能体强化学习的区别，说明合作解谜游戏 AI 中存在分工、协作、非平稳性和稀疏奖励等问题。

## 2. 环境建模

将游戏抽象为多智能体马尔可夫博弈，定义状态空间、观测空间、动作空间、奖励函数和终止条件。

## 3. 方法设计

说明随机策略、规则策略、IPPO 和 MAPPO。重点比较 IPPO 的独立学习方式与 MAPPO 的 centralized training decentralized execution 思想。

## 4. 实验设置

列出地图难度、训练轮数、评价指标和超参数。建议至少比较 basic、key_door、trap 三个关卡。

## 5. 实验结果

展示训练曲线、成功率表格、平均步数表格和 pygame 回放截图。分析 MAPPO 是否比 IPPO 更稳定，以及 agent 是否学到分工。

## 6. 总结与展望

总结项目效果，并讨论通信动作、随机地图、更复杂关卡和人机协作玩法等扩展方向。
